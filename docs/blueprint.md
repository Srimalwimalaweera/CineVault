# **App Name**: CineVault

## Core Features:

- Admin Video Upload: Admin users can upload video thumbnails (portrait size), titles, descriptions, and video links to the platform.
- Firebase Storage: Store video metadata (thumbnail, title, description, link) in a Firestore database.
- Video Card Display: Display videos as cards, including the thumbnail, title, ratings, reaction count, downloads, and views.
- User Authentication: Allow users to sign up and log in to the platform.
- Add to Favorites: Logged-in users can add videos to their favorites list.
- Add to Playlist: Logged-in users can add videos to a custom playlist.
- Detailed Video View: Open video details in a new tab upon card click, showing all metadata.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a sense of sophistication and creativity, reflecting the cinematic content.
- Background color: Very light grey (#F5F5F5), a desaturated version of the primary purple, ensuring a clean and modern backdrop.
- Accent color: A vibrant blue (#2196F3), an analogous color, to highlight interactive elements like buttons and links.
- Font pairing: 'Poppins' (sans-serif) for headlines to give a modern feel, and 'PT Sans' (sans-serif) for body text, providing readability.
- Use clean and simple icons for ratings, reactions, downloads, views, add to favorites, and add to playlist.
- Card-based layout for video display, ensuring responsiveness across devices.
- Subtle animations on card hover and click, providing visual feedback and enhancing user experience.
